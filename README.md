# Инструкция к проекту
В данном проекте 4 папки, у каждой свое назначение:
1) data - папка с датасетами (сырым и очищенным), а также разделение на train, val и test
2) src - основной код проекта, включающий использование LSTM и GPT2 моделей 
3) configs - сохраненные конфигурации, использованные на проекте
4) models - сохраненные веса лучшей модели LSTM

Также прикреплен наглядный график с реализацией моделей, requirements и рекомендация для бизнеса в файле "Результат.txt"
